Fotos de ferias anteriores para Ori
===================================

Carga aqui las imagenes oficiales que Ori puede enviar por WhatsApp a turistas o visitantes.

Formatos aceptados:
- .jpg
- .jpeg
- .png
- .webp

Recomendacion:
- Usa nombres simples, por ejemplo `feria-2026-visitantes-01.jpg`.
- Sube solo fotos aprobadas para uso publico.
- Ori enviara maximo 3 imagenes por conversacion cada 15 minutos para no saturar el chat.
